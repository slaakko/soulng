Project File Grammar

project‑file → PROJECT qualified‑id SEMICOLON declaration*
qualified‑id → ID (DOT ID)*
declaration → reference‑declaration | source‑file‑declaration | text‑file‑declaration | target‑declaration
reference‑declaration → REFERENCE FILEPATH SEMICOLON
source‑file‑declaration → SOURCE FILEPATH SEMICOLON
text‑file‑declaration → TEXT FILEPATH SEMICOLON
target‑declaration → TARGET ASSIGN target SEMICOLON
target → PROGRAM | LIBRARY | UNITTEST
